# 项目一：SKU 缺货率智能监控与自动预警工作流

## 项目概述

速卖通平台 SKU 数量庞大（日均监控 5,000+ 活跃 SKU），商家库存水平参差不齐。
传统人工巡检方式效率低、遗漏多，缺货问题发现滞后导致买家体验受损。

本工作流实现：**实时数据拉取 → 缺货阈值计算 → 智能异常分级 → 自动预警推送 → 看板刷新** 全链路闭环。

## 工作流架构

```
┌──────────────────────────────────────────────────────────────────┐
│                      Cron 定时调度 (每30分钟)                       │
└──────────────────────────────────────────────────────────────────┘
                                │
        ┌───────────────────────┼───────────────────────┐
        ▼                       ▼                       ▼
┌───────────────┐     ┌───────────────┐      ┌───────────────┐
│  data_fetch   │     │anomaly_detect │      │  alert_push   │
│     .py       │────▶│     .py       │─────▶│     .py       │
│  (数据拉取)    │     │  (异常检测)    │      │  (预警推送)    │
└───────────────┘     └───────────────┘      └───────────────┘
        │                       │                       │
        ▼                       ▼                       ▼
┌───────────────┐     ┌───────────────┐      ┌───────────────┐
│  ODPS 数仓    │     │  本地 SQLite   │      │  企业微信群    │
│  (订单/库存表) │     │  (结果缓存)    │      │  钉钉群通知    │
└───────────────┘     └───────────────┘      └───────────────┘
                                │
                                ▼
                      ┌───────────────────┐
                      │   FBI 缺货率看板    │
                      │   (实时刷新)        │
                      └───────────────────┘
```

## AI 工具参与环节

| 环节 | AI 工具 | 具体使用方式 |
|------|---------|------------|
| SQL 查询生成 | Codex | 根据自然语言需求自动生成多表关联 SQL |
| Python 脚本编写 | Codex | 辅助编写数据清洗、异常检测、推送逻辑 |
| 预警文案生成 | Codex | 根据异常等级自动生成差异化推送文案 |
| Bug 调试 | Codex | 快速定位脚本报错并提供修复方案 |

## 实体文件清单

| 文件 | 说明 |
|------|------|
| `sql/sku_daily_monitor.sql` | SKU 维度日度缺货率监控 SQL |
| `sql/merchant_ranking.sql` | 高风险商家排行榜 SQL |
| `sql/category_heatmap.sql` | 品类缺货热力图数据 SQL |
| `scripts/data_fetch.py` | ODPS 数据拉取脚本 |
| `scripts/anomaly_detect.py` | 缺货异常检测与分级脚本 |
| `scripts/alert_push.py` | 企业微信/钉钉预警推送脚本 |
| `dashboard/fbi_dashboard_config.json` | FBI 看板配置 |
| `samples/alert_sample.md` | 预警消息样例 |

## 项目效果

- 缺货发现时效：人工 T+1 → 系统 **< 30 分钟**
- 问题发现效率提升 **30%**，日均自动识别异常 SKU 200+
- 支持 5,000+ SKU 并发监控
