# 审批流通知模板样例

> 四种场景的钉钉/企业微信消息模板，由 Codex 辅助生成文案，人工审核后固化为模板变量。

---

## 场景 1：提交成功通知（→ 申请人）

```
## ✅ 库存修改申请已提交

> 申请编号：**INST-20251210-001**
> 活动：双12年终盛典
> 涉及SKU：2 个 | 合计变更：+150 件
> 提交时间：2025-12-10 10:15
> 预计审批时效：2 小时内

您的申请已进入审批流程，审批结果将通过此消息通知您。
[查看申请详情 →](https://yida.alibaba.com/instance/INST-20251210-001)
```

## 场景 4：审批驳回通知（→ 申请人）

```
## ❌ 库存修改审批未通过

> 申请编号：**INST-20251211-002**
> 审批人：李四（PM）
> 驳回时间：2025-12-11 16:00

> 该SKU的供应商近期产能不足，预计无法在大促前完成补货。

### 🔄 后续步骤
1. 根据驳回原因调整申请内容
2. 重新提交审批申请

[重新提交 →](https://yida.alibaba.com/instance/INST-20251211-002)
```
